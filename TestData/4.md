<p>To use our build system and run our documentation locally, you'll need a copy of Tabler's source files. Follow the steps below:</p>
<ol>
<li><a href="https://nodejs.org/download/" rel="nofollow">Install Node.js</a>, which we use to manage our dependencies.</li>
<li>Navigate to the root <code>/tabler</code> directory and run <code>npm install</code> to install our local dependencies listed in <code>package.json</code>.</li>
<li><a href="https://ruby-lang.org/en/documentation/installation/" rel="nofollow">Install Ruby</a> - the recommended version is <a href="https://cache.ruby-lang.org/pub/ruby/2.5/ruby-2.5.5.tar.gz" rel="nofollow">2.5.5</a>.</li>
<li><a href="https://bundler.io" rel="nofollow">Install Bundler</a> with <code>gem install bundler</code> and finally run <code>bundle install</code>. It will install all Ruby dependencies, such as <a href="https://jekyllrb.com" rel="nofollow">Jekyll and plugins</a>.</li>
</ol>
<p><strong>Windows users</strong>:</p>
<ol>
<li><a href="https://git-scm.com/download/win" rel="nofollow">Install Git</a> in <code>C:\Program Files\git\bin</code> directory and run <code>npm config set script-shell "C:\\Program Files\git\bin\bash.exe"</code> to change the default shell.</li>
<li><a href="https://rubyinstaller.org/downloads/" rel="nofollow">Install Ruby+Devkit</a> - the recommended version is <a href="https://github.com/oneclick/rubyinstaller2/releases/download/RubyInstaller-2.5.5-1/rubyinstaller-devkit-2.5.5-1-x86.exe">2.5.5</a>.</li>
<li><a href="https://jekyllrb.com/docs/installation/windows/" rel="nofollow">Read guide</a> to get Jekyll up and running without problems.</li>
</ol>
<p>Once you complete the setup, you'll be able to run the various commands provided from the command line.</p>
